# Ghalia Aamer - Personal Website

A modern, responsive personal website showcasing Ghalia Aamer's professional profile, projects, and contact information.

## Features

- **Modern Design**: Clean, contemporary design with gradient backgrounds and smooth animations
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: 
  - Smooth scrolling navigation
  - Mobile-friendly hamburger menu
  - Hover effects and transitions
  - Typing animation for hero title
  - Parallax scrolling effects
- **Contact Form**: Functional contact form with validation
- **Social Links**: Integrated social media links
- **Professional Sections**:
  - Hero section with call-to-action
  - About section with skills showcase
  - Projects portfolio
  - Contact information and form

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive features and animations
- **Font Awesome**: Icon library
- **Google Fonts**: Inter font family

## Getting Started

1. **Clone or download** the project files
2. **Open `index.html`** in your browser, or
3. **Run a local server**:
   ```bash
   python3 -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Customization

### Personal Information
Update the following in `index.html`:
- Name and title in hero section
- About section content
- Contact information (email, phone, location)
- Social media links

### Projects
Modify the projects section to showcase your actual work:
- Update project titles and descriptions
- Add real technologies used
- Replace placeholder icons with project images

### Styling
Customize the appearance in `styles.css`:
- Change color scheme (currently purple gradient)
- Adjust fonts and spacing
- Modify animation speeds and effects

### Contact Form
The contact form includes client-side validation. To make it fully functional, you'll need to:
- Add a backend service (Formspree, Netlify Forms, etc.)
- Implement server-side processing
- Add email sending functionality
